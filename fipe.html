Projeto 3: Consumidor da Tabela FIPE

Agora saindo um pouco do âmbito Full-Stack, esse projeto foi criado pra praticar meus conhecimentos na linguagem Java! Depois de alguns meses estudando Python, decidi me aprofundar novamente nesta linguagem que me iniciou no mundo da programação, e como resultado surgiu esse projeto!

É um aplicativo de terminal desenvolvido em Java com o auxílio do framework Spring Boot, que consome a API da Tabela FIPE e exibe informações personalizadas de consultas de automóveis no terminal.

Foi um dos desafios de código mais difíceis que já realizei até hoje, mas um dos mais gratificantes. Foquei em aplicar princípios de boas práticas de programação, ou SOLID, e foi muito interessante poder fazer tudo isso.

Tabela FIPE Consumer

Aplicativo de terminal (CLI) feito em Java e Spring Boot. Consome a API da Tabela FIPE (preços de automóveis) e exibe informações de consulta personalizadas para o usuário, de acordo com o próprio critério de escolha.
