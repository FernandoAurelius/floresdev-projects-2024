Site 1: Jogo do Número Secreto

Agora, estamos chegando na segunda fase da nossa apresentação! A partir daqui, os projetos começam a ser mais focados no desenvolvimento front-end com HTML, CSS e, em alguns casos, JavaScript. O objetivo de cada um deles foi simplesmente pôr em prática meus conhecimentos no "lado claro da força", com designs bonitos e aparências amigáveis.

O primeiro projeto com foco nisso foi o Jogo do Número Secreto, que, ao mesmo tempo, foi o primeiro projeto que fiz esse ano! É um site simples, desenvolvido com HTML, CSS e, principalmente, JavaScript, com foco na Lógica de Programação para criar esse jogo divertido de adivinhação.
